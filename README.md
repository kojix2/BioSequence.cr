# bio-sequence.cr

Function to treat DNA sequence as Slice(UInt8).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     bio-sequence:
       github: kojix2/bio-sequence.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "bio/sequence"
```

## Development
